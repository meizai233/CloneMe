Put your Live2D model files here.

Recommended structure:

- `public/models/<model-name>/<model-name>.model3.json`
- referenced textures and motions in the same folder tree

Then set:

- `VITE_LIVE2D_MODEL_URL=/models/<model-name>/<model-name>.model3.json`
